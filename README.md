## O que é o SeeB

SeeB é uma extensão de navegador e um widget open source focados em acessibilidade web. A ideia central é simples: qualquer página que você acessa pode se tornar mais confortável de ler, independente das suas necessidades visuais ou de leitura.

O projeto nasceu na **ETEC Parque Belém** com a convicção de que tecnologia deve se moldar às pessoas — não o contrário. Cada detalhe foi pensado para simplificar, aproximar e tornar o digital um espaço mais humano.

---

## O problema que resolve

Segundo o WebAIM, **96,3% dos sites têm ao menos um erro de acessibilidade**. No Brasil, apenas **2,9% dos sites** foram aprovados em todos os testes básicos, e somente **0,74%** são totalmente acessíveis segundo os critérios do W3C.

O SeeB não corrige os sites — ele corrige a experiência de quem os usa.

## Duas frentes

### Extensão de Navegador
Um painel de acessibilidade que funciona em qualquer site, ativado diretamente pelo navegador. Disponível para Chrome e todos os navegadores baseados em Chromium.

→ [Instalar na Chrome Web Store](https://chromewebstore.google.com/detail/faoblfaonpejepiealfiogalnkoakdni)

### Widget para Desenvolvedores
Uma única linha de código que integra um painel completo de acessibilidade em qualquer projeto web — HTML estático, React, Next.js, Vue, Angular ou Vite. Sem configuração. Sem coleta de dados. Tudo no navegador.

```html
<script src="https://seeb-widget.pages.dev/widget.js" defer></script>
```

→ [Ver documentação completa](https://seeb-page.vercel.app/docs.html)

## Recursos

- Ajuste de tamanho e espaçamento de fonte
- Filtros de daltonismo (protanopia, deuteranopia, tritanopia e mais)
- Alto contraste e inversão de cores
- Modo de foco para leitura (auxílio para TDAH)
- Ajuste de largura de texto
- Totalmente client-side — nenhum dado é coletado ou enviado
